==========================================================
# ===== SQL - операторы и классы команд ==========
ВСЕ КОМАНДЫ ПИСАТЬ КАПСОМ (БОЛЬШИМИ БУКВАМИ)
==========================================================
DDL - DATA DEFINITION LANGUAGE

Команды для работы (создания и изменения) структуры базы данных
Команды:
    CREATE - создание
    ALTER - изменение
    DROP - удаление
    TRUNCATE - очистка таблицы
===========================================================
DML - DATA MANIPULATION LANGUAGE
Команда для работы с данными

Команды:
    INSERT - добавить данные
    UPDATE - обновить(изменить) данные
    DELETE - удаление данных
==========================================================
DQL - DATA QUERY LANGUAGE
Команды для получения данных

Команды:
    SELECT - выборка дыннх(получение)
==========================================================
DCL - DATA CONTROL LANGUAGE
Команды для управления правами доступа

Команды:
    GRANT - Выдвть права
    REVOKE - Забираем права
==========================================================
TCL - TRANSACTION CONTROL LANGUAGE
Команды для управления транзакциями (ВСЕ ЛИБО НИЧЕГО)

Команды:
    COMMIT - сохранения изменений
    ROLLBACK - отмена изменений
    SAVEPOINT - точка сохранения
==========================================================
